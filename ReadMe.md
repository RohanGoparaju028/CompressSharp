### CompressSharp 
CompressSharp is a compression library that compresses large dataset stored in S3.The library is primarily writtern in C#.
The library uses the individiual AWS credential and works with the dataset you are accessed to.
The library has 3 functinalites 
1) Authenticate and Aurtorize the users credentials
2) Find the dataset that you wanna compress
3) Compress and install the dataset in the current directory from which the library is being used.
